# 🏦 Basic Banking System (Java)

![Java](https://img.shields.io/badge/Java-OOP-orange)
![Status](https://img.shields.io/badge/Status-Active-success)
![Project Type](https://img.shields.io/badge/Project-Console_App-blue)

A simple **console-based banking system** built using Java that simulates core banking operations such as account creation, deposits, withdrawals, and balance checking.  
This project focuses on practicing **Object-Oriented Programming (OOP), modular design, and transaction management**.

---

## 📌 Project Overview

This project demonstrates how basic banking functionalities can be implemented using a **Java command-line interface (CLI)** application.

The system allows users to:

- Create a bank account
- Deposit money
- Withdraw money
- Check account balance
- View basic account details

The goal of this project is to strengthen understanding of **Java fundamentals, classes, objects, and method-driven program design**.

---

## 🛠 Tech Stack

- **Language:** Java  
- **Concepts Used:**  
  - Object-Oriented Programming (OOP)  
  - Classes and Objects

## ▶️ How to Run

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/IconicN1nja/baisc-banking-system.git
````

### 2️⃣ Navigate to the Project Folder

```bash
cd baisc-banking-system
```

### 3️⃣ Compile the Program

```bash
javac *.java
```

### 4️⃣ Run the Program

```bash
java BankingSystem
```

## 📚 Concepts Demonstrated

* **Encapsulation** – Protecting account data inside classes
* **Method Abstraction** – Separating banking operations into reusable methods
* **Control Flow** – Menu-driven program using loops and conditional statements
* **Input Handling** – Reading user input using Java `Scanner`
* **Transaction Validation** – Preventing invalid withdrawals or deposits
